# Changelog

All notable changes to the **SAGE Studio Agent** VS Code extension will be documented here.

## [0.1.0] — 2026-03-04

### Added
- Chat panel with SSE streaming and AgentStep rendering (thinking, tool_call, tool_result, response)
- Agent Task panel for submitting multi-step tasks to the sage-studio AgentOrchestrator
- Studio backend start/stop via configurable shell commands
- Status bar indicator with live /health polling
- Context menu: "Open Chat" / "Open Agent Task Panel" from code selection
- Installation Guide webview
- VS Code settings for host, port, TLS, system prompt, start/stop commands
